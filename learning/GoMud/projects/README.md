# 历史脉络实战项目

> 每个阶段 = 概念脉络 + 历史约束 + 实战项目
> 只用"那个时代已经存在"的概念，站在当时程序员的鞋子里解决真问题

## 阶段 × 项目 总览

### #1 变量与内存（1957 Fortran 时代）
- **可用**：变量、基本类型（int/string/bool）、运算符、赋值、fmt.Println/Printf
- **没有**：if/for、函数、struct、指针、多返回值
- **历史背景**：1957 Fortran 是第一个广泛使用的高级语言，"变量"刚被发明。程序员终于不用手写机器码/汇编，能用名字代替地址。但只有基本类型和运算，连 if 都很原始（Fortran 的 IF 是算术 IF，三路跳转）
- **当时真实需要**：科学计算（弹道、物理）、工程换算、数据初始化
- 项目：
  - [P1-1] 找零计算器：给定价格和付款，算找零（硬写，无循环）
  - [P1-2] 体温换算：摄氏↔华氏↔开尔文三向转换（硬写三段）
  - [P1-3] 角色属性初始化：用变量存 RPG 角色 Name/HP/MaxHP/Level/Exp，打印角色卡

### #2 控制流 if/for（1968 结构化编程时代）
- **可用**：+ if/else、for（含 range）、break/continue
- **没有**：函数、struct、指针、多返回值
- **历史背景**：1968 Dijkstra "Go To Considered Harmful"，结构化编程确立 if/for 三种基本结构。之前 Fortran 的 GO TO 跳得到处都是，程序没法维护
- **当时真实需要**：批处理数据、表格生成、简单分类统计
- 项目：
  - [P2-1] 体温分级：根据体温打印"正常/低烧/中烧/高烧/送医"
  - [P2-2] 九九乘法表：双层 for 打印
  - [P2-3] 找零计算器 v2：用 for 遍历面额数组，替代 v1 硬写

### #3 函数（1950s Fortran/汇编时代）
- **可用**：+ 函数、参数、单返回值
- **没有**：多返回值、struct、指针
- **历史背景**：函数 1950s 就有了（Fortran FUNCTION/SUBROUTINE），比 if/for 结构化还早。解决"重复逻辑写十遍"的痛点
- **当时真实需要**：数学函数库、科学计算子程序、代码复用
- 项目：
  - [P3-1] 体温换算函数库：把 #1 的三向转换抽成三个函数
  - [P3-2] 质数判断：写 isPrime(n) 函数
  - [P3-3] 简易计算器：add/sub/mul/div 四个函数 + if dispatch

### #4 多返回值 & 命名返回值（Go 特色，无直接历史对应）
- **可用**：+ 多返回值、命名返回值、`_`、逗号 ok 模式
- **没有**：struct、指针
- **历史背景**：这是 Go 独有的设计，C 用指针参数塞结果，Java 用包装对象。Go 直接在栈上多开格子
- **当时真实需要**：函数要同时返回"结果 + 状态"
- 项目：
  - [P4-1] 整除带余数：divmod(a,b) 返回 (商, 余数)
  - [P4-2] 安全开方：sqrt(n) 返回 (结果, error)，负数返回 error
  - [P4-3] 字符串查找：find(haystack, needle) 返回 (位置, ok)

### #5 指针（1972 C 时代）
- **可用**：+ 指针、`&`、`*`、指针参数、struct（无方法）
- **没有**：方法、interface
- **历史背景**：1972 C 语言，Dennis Ritchie 选了"默认拷贝"，发明指针作为补偿。struct 在 C 里早就有了（1970s 初），但方法还没有
- **当时真实需要**：链表/树等数据结构、原地修改数据、swap 函数
- 项目：
  - [P5-1] swap 函数：经典 C 入门题，指针参数实现 swap
  - [P5-2] 链表节点：Node struct + 指针串联 3 节点 + 遍历打印
  - [P5-3] 原地修改数组：函数参数 `*[]int`，把所有元素乘以 2

### #6 Struct/方法/继承（1967 Simula 时代）
- **可用**：+ struct 方法、内嵌
- **没有**：interface
- **历史背景**：1967 Simula 发明 class + 方法 + 继承，把数据和操作绑一起。Go 用 struct + 方法 + 内嵌实现 has-a
- **当时真实需要**：模拟系统（Simula 就是 Simulation Language）、银行账户、几何图形
- 项目：
  - [P6-1] 银行账户：Account struct + 存款/取款方法，指针接收者改余额
  - [P6-2] 几何图形：Point struct + Distance 方法；Rectangle 内嵌 Point + Area 方法
  - [P6-3] 学生成绩单：Student struct + 添加成绩 + 计算平均分方法

### #7 Interface（1970s Smalltalk 鸭子类型）
- **可用**：+ interface、动态分派
- **没有**：error 接口（还没教）
- **历史背景**：1970s Smalltalk 反抗继承僵硬，"不关心你是什么，只关心你能做什么"。Go 用隐式接口 + 两格结构高效实现
- **当时真实需要**：多态、可替换组件、通用算法
- 项目：
  - [P7-1] 形状面积计算器：Shape interface{ Area() float64 }，Circle/Rectangle/Triangle 实现，一个函数算总面积
  - [P7-2] 动物叫声：Animal interface{ Sound() string }，Dog/Cat/Cow 实现，遍历打印
  - [P7-3] 通用排序：Sorter interface{ Less(i,j int) bool; Swap(i,j int); Len() int }，写一个通用 bubble sort
